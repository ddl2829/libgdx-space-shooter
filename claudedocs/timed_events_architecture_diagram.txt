## Visual Marker Rendering Algorithm

For each TimedEvent in level.timedEvents:

    1. Calculate Y position
       ├─ If POSITION_BASED: yPos = triggerTime
       └─ If TIME_BASED: yPos = (triggerTime * scrollSpeed)

    2. Adjust for viewport scroll
       adjustedY = yPos - scrollOffsetY

    3. Check visibility
       If adjustedY outside viewport: skip rendering

    4. Get marker color by event type
       ├─ NOTIFICATION: Blue
       ├─ ITEM_SPAWN: Green
       ├─ ENEMY_WAVE: Red
       └─ ENVIRONMENTAL_CHANGE: Purple

    5. Draw marker components
       ├─ Background circle (colored)
       ├─ Icon emoji (white text)
       ├─ Dashed connector line
       ├─ Trigger label (time/position)
       └─ Event summary (truncated)
